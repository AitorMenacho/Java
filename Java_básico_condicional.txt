//En este ejemplo lo que harémos será comprobar si la persona es mayor de edad o menor. En el caso de España para que una persona sea mayor de edad tiene que superar los 18 años.

Algoritmo edad

	Definir edad como entero;
	
	edad = 19 //He usado el signo "=" pero de momento significa lo mismo que "<-"

	//Ahora usaremos la expresión Si ... Entonces ejecutamos una acción.

	Si edad > 18 Entonces
		Escribir "Eres mayor de edad";
	SiNo //Esto se ejecuta en caso de que la persona sea menor del número que pusimos delante del "Entonces".
		Escribir "Eres menor de edad";
	Fin Si

	//Con esto básicamente lo que hacemos es decirle que si se cumple una condición (en este caso que sea mayor de 18 años) ejecute una acción, pero en caso contrario ejecute otra.

	//Hay que tener en cuenta que al poner el simbolo ">" le estamos diciendo que la variable "edad" tiene que ser mayor a 18, por lo que si la variable "edad" contiene un 18 saltará como si fueras menor.
	  Para solucionar eso añadiremos a ">" un "=" justo delante.

	Si edad >= 18 Entonces
		Escribir "Eres mayor de edad";
	SiNo //Esto se ejecuta en caso de que la persona sea menor del número que pusimos delante del "Entonces".
		Escribir "Eres menor de edad";
	Fin Si
	
	//Con esto le decimos que si tienes 18 o más eres mayor de edad y en caso contrario menor.

	//Esto mismo puedes hacerlo sin el uso del "SiNo" pero si la persona es menor de 18 no saltará ningún aviso, simplemente no se ejecutará.

FinAlgoritmo
